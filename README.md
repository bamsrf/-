# Вертушка 🎵

Мобильное приложение для любителей винила — ведите коллекцию пластинок, отслеживайте стоимость, делитесь вишлистом с друзьями.

---

## Что умеет приложение

### Коллекция
- Добавляйте пластинки через поиск по названию, артисту или лейблу
- **Сканируйте штрихкод** на обложке — приложение найдёт пластинку автоматически
- **Распознавание обложки** через камеру — направьте на пластинку, и она добавится сама
- Организуйте коллекцию по папкам (например: «Джаз», «Подарки», «Продать»)
- Смотрите **оценочную стоимость** всей коллекции в рублях по актуальному курсу

### Поиск
- Поиск по базе Discogs — более 15 миллионов релизов
- Карточки альбомов с обложками, годом, страной и форматом
- Страницы артистов со всей дискографией

### Вишлист
- Список пластинок, которые хотите купить
- **Поделитесь ссылкой** — друзья смогут увидеть ваш вишлист и забронировать подарок
- Перенос пластинки из вишлиста в коллекцию одной кнопкой

### Социальное
- Публичный профиль с вашей коллекцией
- Подписывайтесь на других коллекционеров
- Лента активности — смотрите что добавляют те, на кого подписаны

### Прочее
- Экспорт коллекции и вишлиста в CSV (для Excel и таблиц)
- Онбординг для новых пользователей

## Технологии

### Мобильное приложение
- **React Native** + **Expo** (кроссплатформенно: iOS и Android)
- **TypeScript** — строгая типизация
- **Expo Router** — навигация на основе файлов
- **Zustand** — управление состоянием
- **expo-image** — кэширование изображений на диске
- **Axios** — HTTP-клиент с retry и дедупликацией запросов

### Сервер (Backend)
- **Python** + **FastAPI** — быстрый асинхронный API
- **PostgreSQL** — основная база данных
- **Redis** — кэш для быстрых ответов (переживает рестарт сервера)
- **SQLAlchemy** — работа с базой данных
- **APScheduler** — фоновые задачи (обновление цен, очистка кэшей)
- **Docker** + **Docker Compose** — запуск в контейнерах

### Внешние сервисы
- **Discogs API** — база данных всех пластинок мира
- **OpenAI Vision** — распознавание обложек через камеру
- **ЦБ РФ** — актуальный курс доллара для стоимости коллекции

## Структура проекта

```
Вертушка/
├── Backend/          # Сервер (FastAPI)
│   ├── app/
│   │   ├── api/      # Эндпоинты: auth, records, collections, wishlists, users

│
└── Mobile/           # Приложение (React Native / Expo)
    ├── app/          # Экраны (Expo Router)
    │   ├── (auth)/   # Вход, регистрация, сброс пароля
    │   ├── (tabs)/   # Главные вкладки: главная, коллекция, поиск
    │   └── ...       # Детали записи, артиста, папки, профиль
    ├── components/   # Переиспользуемые компоненты
    ├── lib/          # api.ts, store.ts, types.ts
    └── constants/    # Тема и цвета
```

## Запуск локально

### Backend

```bash
cd Вертушка/Backend

# Создайте .env файл на основе .env.example
cp .env.example .env

# Запуск через Docker (рекомендуется)
docker-compose up -d

# Или напрямую
pip install -r requirements.txt
uvicorn app.main:app --reload
```

API будет доступно на `http://localhost:8000`

### Mobile

```bash
cd Вертушка/Mobile
npm install
npm start
```

Откройте в Expo Go на телефоне или запустите симулятор.

> Для работы с локальным сервером укажите ваш IP в [lib/api.ts](Mobile/lib/api.ts) в переменной `API_BASE_URL`.
