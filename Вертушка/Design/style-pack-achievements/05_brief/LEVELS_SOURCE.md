# Вертушка — Система уровней пользователя

## Концепция

Пользователь копит XP-очки через ачивки и растёт по лестнице из 10 уровней-архетипов.
Уровни не привязаны к конкретным сериям ачивок — это чистая XP-прогрессия.
Тематика набора: **физика звука** — пользователь проходит путь от тишины до первозвука.

## Контекст для реализации

- Флейвор-текст показывается **в момент повышения уровня** (уведомление / экран поздравления).
- Иконка каждого уровня — стиль **эмалевого пина**, отражает акустический образ названия.
- Уровень отображается в профиле пользователя как «звание коллекционера».
- Серий ачивок 10+: foundation, scale, genres, geography, eras, discography, rarity, community, gifts, invitations, + скрытые пасхалки (73+ ачивки всего).
