# План: промпт для Claude Design — 10 пинов Phase 0

## Context

В [PLAN_ACHIEVEMENTS_V2.md](../../Desktop/Cursor/Вертушка/docs/plans/PLAN_ACHIEVEMENTS_V2.md) §10.1 заявлен каталог Phase 0 — 10 ачивок, которые открывают live-режим системы ачивок Вертушки. По §3.2 каждая ачивка рендерится как **эмалевый пин со своим сюжетом**, и сейчас в коде лежат только живые SVG-плейсхолдеры (`Mobile/components/achievement-scenes/index.tsx`, ~860 строк) — финальные пины ждут дизайнера.

Задача — собрать **единый бриф для Claude Design**, чтобы они вернули первые драфты всех 10 пинов одним пакетом, согласованных между собой по стилю, материалу и палитре. Без брифа драфты разъедутся, и придётся согласовывать каждый отдельно.

Решения, принятые до написания промпта:

1. **Финиш — soft enamel** (текстурно, с глубиной): эмаль утоплена ниже металлических линий, видны рельефные канты, есть блики. Богаче на 256 / 700 px, но требует от художника аккуратной работы со светом, чтобы при 64 px (грид) пины не тускнели.
2. **Тир кодируется ауры, не самим пином** (Tyler/IGOR-стиль): каждый пин имеет свою палитру под метафору, тир-цвет (`#A5C8E1 / #5B7DD8 / #E89AC0 / #1B237D / #0A0A1A`) выражается **через halo-ауру позади пина** и лёгкий sparkle для 🌸+. Контур у всех — золото `#D9A84E` + тёмный outline `#0B1438`. Это даёт каждому пину собственный характер, как в референсах пользователя (Mokita green set, UFO welcome, Tyler the Creator).
3. **Названия Phase 0 как в плане V2**, метафоры — по §3.2. Для A4 пользователь предложил «Окно нараспашку» — внесём как nickname-вариант, но основное название оставим «Распахнул» (короче в UI и в тосте анлока).

Промпт ниже — единый блок, готовый к копипасте в Claude Design. После того как драфты вернутся, дизайнер сравнивает их с палитрой Вертушки (T.palette в `Mobile/constants/theme.ts`) и плейсхолдерами в `achievement-scenes/index.tsx`, итерирует точечно по проблемным пинам.

## Финальный промпт для Claude Design

> Копируется блоком ниже целиком. Все хексы и размеры выверены под существующую систему `Mobile/constants/theme.ts` и `Mobile/components/AchievementPin.tsx`.

```text
Роль: ты дизайнер-иллюстратор, рисуешь набор эмалевых пинов для приложения «Вертушка» — это коллекционер винила (Expo/React Native). Нужны первые драфты 10 пинов системы ачивок (Phase 0). Все 10 должны выглядеть как один цельный сет — единый материал, контур и масштаб деталей. Драфты потом доводятся, поэтому важнее цельность серии и точная метафора, чем финальный полишинг.

═══════════════════════════════════════════════════════
МАТЕРИАЛ И СТИЛЬ
═══════════════════════════════════════════════════════

Имитируем soft enamel pin:
• Металлический контур приподнят над эмалью (раздельные «бортики» между цветными зонами).
• Эмаль внутри зон утоплена, на поверхности — лёгкий specular highlight (тонкая дуга белого блика вверху каждой крупной заливки).
• По внешнему краю пина — золото `#D9A84E` (3–4% площади) с тонким тёмным outline `#0B1438` снаружи (1px), чтобы пин читался на любом фоне.
• Внутри плоскостей золотая линия 1.5–2 px разделяет эмалевые зоны. Эта линия — главный носитель деталей.
• Скрытая иерархия: крупные эмалевые зоны → металлические линии-разделители → мелкие акценты (искры, блики, маленькие надписи).

Референсы стиля (визуальная мудбордная база):
1. **Mokita green pin set** — карточные шахматные мотивы, золото-чёрный outline, насыщенная одна эмаль, разные силуэты в одном наборе.
2. **UFO “WELCOME” pin** — круглый формат с одной плотной эмалью и графичным сюжетом внутри.
3. **Tyler the Creator IGOR / Flower Boy / If You Get Lost set** — каждая шайба имеет свою палитру под обложку, но единый «винтажно-медальный» язык.

Что НЕ берём из референсов:
• Никаких реалистичных фото-теней — это иллюстрация эмали, не 3D-рендер.
• Без gradient mesh, без noise-фильтров. Только плоская заливка + одна линия блика + один inner shadow по верхней дуге.
• Без подписей-слоганов внутри пина (кроме случаев, когда цифра — часть метафоры: «10», «50», «33⅓»). Названия живут в UI, не на пине.

═══════════════════════════════════════════════════════
ФОРМА (силуэт)
═══════════════════════════════════════════════════════

• Каждый пин — свой силуэт. Никогда все 10 одного абриса.
• Сюжет может выходить за прямоугольник 100×100 (мачта корабля, шторы окна, бант коробки, лучи пластинки) — это часть стиля.
• Пин должен распознаваться при 64×64 px (грид-карточка в приложении). Это значит: одна крупная узнаваемая фигура занимает ≥55% площади, мелкие детали жертвуются в пользу силуэта.
• Целевые размеры рендера: основной — 700×700 px (для share-card), уменьшается до 256 (анлок-анимация) и 64 (грид).

═══════════════════════════════════════════════════════
ПАЛИТРА И ТИР
═══════════════════════════════════════════════════════

Брендовая ось приложения — navy / cobalt + тёплый ember-акцент. Используй из этого:
• Navy `#0B1438` — основной тёмный, для outline и силуэтов.
• Cobalt `#2A4BD7` — насыщенный синий, акценты.
• Cobalt Soft `#5C7AE8` — мягкий синий для крупных заливок.
• Ember `#E85A2A` — тёплый оранжево-красный, ТОЛЬКО для центральных лейблов пластинок и пары sparkle-точек.
• Ivory `#FBF5EA` — кремовый, «бумага» / лента / фон портрета.
• Gold `#D9A84E` — единый цвет металлической линии и контура у всех 10.
• Vinyl Black `#1A1A2E` — глубокий тёплый чёрный для самих пластинок.

ТИР НЕ КРАСИТ ПИН ВНУТРИ — он выражается **аурой-halo позади** (рендерится отдельно в UI, тебе её рисовать не нужно, но визуально учитывай совместимость):
• 💧 Простая → halo `#A5C8E1` светло-голубой
• 🔵 Заметная → halo `#5B7DD8` насыщенный голубой
• 🌸 Редкая → halo `#E89AC0` розовый + 2–3 sparkle-точки на самом пине
• 🌌 Эпическая (не в Phase 0)
• ⚫ Легенда (не в Phase 0)

У 🌸-пинов (#9 и #10) внутри пина — 2–3 крошечные звёздочки-искры золотом, как тонкий намёк на редкость. У 💧 и 🔵 — без искр.

Метафора: поставил лицо — портрет-карточка эпохи 78 RPM.
Сюжет: овальный медальон с фигурным золотым обрамлением (фестоны/завитки), внутри — обобщённый силуэт головы (плечи + овал головы, без лица), на месте лица — мини-лейбл пластинки. Снизу медальона — лента-баннер, пустая (имя дорисовывается в UI).
Силуэт: овал + лента, концы ленты торчат за левый и правый края рамки.
Палитра: медальон ivory `#FBF5EA`, силуэт навай `#0B1438`, обрамление золото `#D9A84E`, лента cobalt `#2A4BD7` с золотыми концами.

──── #4. A4 «Распахнул» — Простая 💧 ────
Метафора: открыл коллекцию миру — окно нараспашку, из него вылетают носители.
Сюжет: прямоугольное окно с арочной верхушкой, рамы расчерчены на 4 секции (как классическое окно), две створки распахнуты вправо-влево; за окном виден силуэт ночного города, в котором здания собраны из аналоговых форматов (башня-кассета с двумя бобинами-окнами, купол-диск, антенна-игла, дом-конверт пластинки). На переднем плане 3 пластинки в полёте, разлетающиеся из окна, как листы на ветру.
Силуэт: створки окна выступают за левую и правую границу рамки, одна летящая пластинка торчит за верхнюю.
Палитра: небо за окном бледно-голубой `#A5C8E1`, силуэт-город navy `#0B1438`, рамы и переплёт — золото `#D9A84E`, пластинки в полёте vinyl black `#1A1A2E` с ember-лейблами `#E85A2A`. Один маленький месяц или звезда на небе золотом.

Палитра: парус cobalt `#2A4BD7` с ivory `#FBF5EA` конвертом, корпус-пластинка чёрный `#1A1A2E` с золотыми канавками и ember `#E85A2A` лейблом, волны cobalt soft `#5C7AE8`, мачта и флагшток — золото. Это «выпускной» пин серии, поэтому слегка богаче деталями.

Палитра: 10 корешков в разной палитре из бренда (cobalt, ember, ivory, cobalt soft, navy, и нейтральные — повторяться можно, но не подряд). Полка-основание — navy `#0B1438`. Цифра — золото с inner shadow navy.
```

## Verification

После того как Claude Design вернёт драфты:

1. Положить 10 SVG в `Mobile/assets/achievements/` и contact-sheet в `Design/screens&covers/achievements/`.
2. Открыть contact-sheet рядом с тремя пользовательскими референсами — проверить:
   - Все 10 одного финиша (soft enamel) и одной толщины золотой линии.
   - При уменьшении до 64 px (через preview) силуэт каждого пина читается с расстояния.
   - Тир-нейтральность: при наложении трёх halo-цветов (`#A5C8E1` / `#5B7DD8` / `#E89AC0`) ни один пин не «теряется» — выбранная палитра (Tyler-стиль) даёт контраст с любой аурой.
3. Подменить два-три плейсхолдера в `Mobile/components/achievement-scenes/index.tsx` (например, `ScenePoehali`, `SceneRaspakhnul`) и запустить `cd Mobile && npm start`, проверить рендер в гриде `/achievements` и в `AchievementUnlockOverlay` на iOS-симуляторе.
4. Если хоть один пин не читается в 64 px → отдельная итерация только по этому пину, не перерисовываем весь сет.
