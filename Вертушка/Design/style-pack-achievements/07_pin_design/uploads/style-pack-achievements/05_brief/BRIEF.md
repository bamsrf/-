# Brief: экран «Ачивки» — переработка под концепт «Витрина пинов-коллекционера»

## Контекст продукта
**Вертушка** — мобильное приложение для коллекционеров виниловых пластинок (iOS/Expo).
Айдентика B1 «Stamper Hi-Fi» v2: navy + cobalt + ember + ivory. Эстетика — винтажный
коллекционный пин-арт, эмалевые значки, винил-культура 70-80-х в современной обёртке.

## Текущая боль (см. `06_current_state/`)
1. Серые круги-плейсхолдеры locked-пинов выглядят дёшево и плоско.
2. Меты-ачивки (главные трофеи серии) выделены только маленькой золотой звёздочкой,
   теряются в общем гриде.
3. Серии живут на светлой surface-карточке (#F0F2FA), пины не «горят» как трофеи.
4. Блок «Пасхалки» (бывшие «Сюрпризы», переименованы) не выделен визуально —
   воспринимается как обычная серия.
5. Hero-карточка плоская, нет ощущения витрины/трофея.

## Метафора нового экрана
**Бархатная витрина коллекционера эмалевых пинов.** Каждая ачивка — отдельный
enamel pin: золотой rim, навая enamel-вставка, плоская иллюстрация в стиле винтажных
коллекционных значков. Locked-пины — выгравированные пустые гнёзда с замочком, не
серые круги.

См. референсы пинов **«100», «500», «vinyl-box»** (приложены отдельно к чату) —
это **точный target-стиль** для каждого пина.

## Иерархия экрана

### 1. Hero-витрина (360pt)
- Фон: gradient `brand` (navy → cobalt → cobaltSoft) + grain texture 4%
- Слева (60%): крупный пин архетипа (текущий — «Услышали» / ухо) в стиле enamel
- Справа: «4/71» **Rubik Mono One 56pt** ivory + подпись «ачивок открыто»
- Капсула архетипа «Новичок» (h3 navy на ivory, gold outline 1pt, radius.full)
- Внизу: золотая капсула «🥚 Пасхалки · 1» (small)

### 2. ВИТРИНА МЕТ-ТРОФЕЕВ (новая секция)
**Заголовок:** «🏆 Главные трофеи» + counter «2/8» (золотая капсула)
- Грид 2-в-ряд, ячейка 160×200pt
- Подложка ячейки: navy `bg.deep`, radius.lg, gold border 2pt
- Enamel-пин 120pt по центру
- Под пином — название h3 ivory
- **Locked:** gold ободок остаётся, эмаль пустая (silhouette гравировкой), замочек
- **Unlocked:** full-color enamel + ember glow shadow (blur 24, opacity 0.4)
- **Hover (press):** vinylSpin/8 + emberGlow appear

### 3. СЕРИИ (повторяющийся блок ×6)
- Карточка серии: **navy `bg.deep`** (не светлый surface!) radius.xl
- Тиснёный grooves-паттерн на подложке (концентрические круги ivory.06)
- Header: emoji 32pt + title h1 ivory + gold capsule counter «3/6»
- Description: ivory.60 14pt
- Грид пинов 3-в-ряд, ячейка 100pt, gap 16pt
- **Pin states (80pt):**
  - Locked normal: gold outline 1.5pt, silhouette тонкой линией, lock-emoji
  - Locked progress ≥0.75: pulsing gold rim (2s loop)
  - Unlocked: full enamel + thin gold outline
  - Meta (последний): размер 96pt, gold outline 2.5pt, label жирный
- Прогресс «7/10»: золотая capition warning ниже пина

### 4. БЛОК «🥚 ПАСХАЛКИ» (особая подача — «секретный сейф»)
- Карточка: navy `bg.deep` + ember glow border 1pt + emberSoft inner glow
- Header: «🥚 Пасхалки» h1 ivory + ember capsule «1 открыто»
- Description italic: «Их находят сами. Не подсматривай.»
- Открытые пасхалки: грид как у серий, но ember-rim вместо gold
- Empty state: треснувшее золотое яйцо silhouette + текст «Пока ничего не нашлось»

## Визуальный стиль каждого пина (КРИТИЧНО)
- Замкнутый контур gold (`#A86614 → #E85A2A` gradient на rim)
- Внутренняя сцена в navy/cobalt/ember палитре
- Плоская заливка с тонкими ivory highlights (enamel-блики)
- Высокий контраст: navy фон значка + ivory/ember иллюстрация
- Никаких градиентов внутри сцены — только плоские плашки и чёткие линии
- См. референсы 100/500/vinyl-box в чате — это база

## Что НЕ делать
- Серые круги для locked-states
- Звёзды для мет (мета = больше + двойной gold rim)
- Светлый surface фон карточек серий
- Полупрозрачные/блюрные элементы (это не glassmorphism)
