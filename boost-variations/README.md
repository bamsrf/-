## Технологии

- **HTML5** - семантическая разметка
- **CSS3** - современные фичи (CSS Variables, Grid, Flexbox, backdrop-filter)
- **Vanilla JavaScript** - без фреймворков
- **Google Fonts** - Inter, Space Grotesk

## Локальный запуск

Просто откройте любой `index.html` в браузере:

```bash
# Открыть Theme 1
open theme-01-minimalism/index.html

# Открыть Theme 2
open theme-02-glassmorphism/index.html

# Открыть Theme 3
open theme-03-neubrutalism/index.html
```

Или используйте live server:
```bash
# С помощью VS Code Live Server расширения
# Или с помощью Python
cd theme-01-minimalism
python3 -m http.server 8000
```

## Деплой на Beget

Смотрите детальную инструкцию в [DEPLOY_BEGET.md](./DEPLOY_BEGET.md)

**Быстрый старт:**
1. Создайте поддомены v1, v2, v3 в панели Beget
2. Загрузите файлы в соответствующие папки
3. Исправьте путь к `script.js` в каждом HTML
4. Готово!

## Особенности реализации

### Общий JavaScript
Все темы используют один `script.js`:
- Smooth scroll для якорных ссылок
- FAQ аккордеон
- Scroll эффект для header
- Intersection Observer для анимаций при скролле

### CSS Variables
Каждая тема использует CSS переменные для легкой кастомизации:
```css
:root {
    --primary: #2B5CE6;
    --secondary: #000000;
    --background: #FFFFFF;
    /* и т.д. */
}
```

### Адаптивность
Все темы полностью адаптивные:
- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: < 768px
