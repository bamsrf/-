# Инструкция по деплою на Beget

## Обзор проекта

Созданы 3 визуальные темы для лендинга BOOST Intensive:
- **Theme 1**: Pure Minimalism (белый, черный, синий акцент)
- **Theme 2**: Glassmorphism Blue (стекло, blur эффекты, градиенты)
- **Theme 3**: Neubrutalism Bright (яркие цвета, толстые границы, bold)

## Структура файлов

```
boost-variations/
├── theme-01-minimalism/
│   ├── index.html
│   └── style.css
├── theme-02-glassmorphism/
│   ├── index.html
│   └── style.css
├── theme-03-neubrutalism/
│   ├── index.html
│   └── style.css
└── shared/
    └── script.js (используется всеми темами)
```

## Шаг 1: Подготовка файлов

1. Убедитесь, что все файлы созданы и работают локально
2. Проверьте, что пути к `script.js` правильные во всех HTML файлах

## Шаг 2: Настройка поддоменов на Beget

### 2.1 Войдите в панель управления Beget
- Перейдите на https://cp.beget.com/
- Войдите в свой аккаунт

### 2.2 Создайте поддомены

**Для Theme 1 (Minimalism):**
1. Перейдите в раздел "Домены" → "Поддомены"
2. Нажмите "Добавить поддомен"
3. Введите имя: `v1`
4. Выберите основной домен: `boost-intensive.ru`
5. Укажите папку: `/public_html/v1` (или создайте новую)
6. Сохраните

**Для Theme 2 (Glassmorphism):**
1. Повторите процесс
2. Имя поддомена: `v2`
3. Папка: `/public_html/v2`

**Для Theme 3 (Neubrutalism):**
1. Повторите процесс
2. Имя поддомена: `v3`
3. Папка: `/public_html/v3`

## Шаг 3: Загрузка файлов через FTP

### 3.1 Подключитесь к FTP
Вы можете использовать:
- FileZilla (https://filezilla-project.org/)
- Встроенный файловый менеджер Beget

**FTP данные (найдите в панели Beget):**
- Хост: `boost-intensive.ru` (или FTP-адрес из панели)
- Логин: ваш логин Beget
- Пароль: ваш пароль FTP
- Порт: 21

## Шаг 4: Проверка работоспособности

Откройте в браузере:
- https://v1.boost-intensive.ru/ - должна открыться минималистичная тема
- https://v2.boost-intensive.ru/ - стеклянная тема с blur
- https://v3.boost-intensive.ru/ - яркая брутальная тема

## Шаг 5: Настройка DNS (если поддомены не работают)

Если поддомены не открываются сразу:
1. Подождите 15-30 минут (DNS обновление)
2. Проверьте настройки DNS в панели Beget
3. Убедитесь, что A-записи указывают на IP вашего сервера

## Альтернативный способ: через файловый менеджер Beget

Если у вас нет FTP-клиента:

1. Войдите в панель Beget
2. Перейдите в "Файловый менеджер"
3. Создайте папки: `v1`, `v2`, `v3` в `/public_html/`
4. Загрузите файлы через веб-интерфейс:
   - Нажмите "Загрузить"
   - Выберите нужные файлы
   - Загрузите в соответствующие папки

## Возможные проблемы

### Поддомен не открывается
- Подождите 15-30 минут для обновления DNS
- Проверьте правильность настройки поддомена
- Убедитесь, что файлы загружены в правильную папку

### Стили не применяются
- Проверьте путь к style.css в index.html
- Убедитесь, что файл style.css загружен
- Проверьте права доступа к файлам (644 для файлов)

### JavaScript не работает
- Проверьте путь к script.js
- Откройте консоль браузера (F12) и проверьте ошибки
- Убедитесь, что файл script.js загружен

### Шрифты не загружаются
- Шрифты загружаются с Google Fonts, требуется интернет
- Проверьте, что ссылки на Google Fonts в &lt;head&gt; правильные
